Issue #139 checkpoint

Tools survive HTTP decode and gateway orchestration. Silent failure occurs at provider translation: native structs omit tools/tool_choice; slim OpenAI-compat wrappers omit top-level tools; OpenAI streaming drops assistant ToolCalls in buildMessages and omits tool_calls deltas. MCP loop never fires when resp.Choices[].ToolCalls is empty → text-only response, no error.

Probe order: decode → pre-provider req → outbound HTTP body → raw API response → StreamChunk.Delta.ToolCalls
